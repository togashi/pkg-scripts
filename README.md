# pkg-scripts

```shell
Usage: pkg-scripts [options] [command]

script manager for package.json

Options:
  -V, --version   output the version number
  -h, --help      display help for command

Commands:
  list|ls         list scripts
  has             check existency
  put|add         put script
  rename|mv       rename script
  remove|rm       remove script
  copy|cp         copy script
  help [command]  display help for command
```
